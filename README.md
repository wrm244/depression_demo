## 全球抑郁症数据大数据分析bigdata_depressiond项目前端

### 项目介绍

该项目纯``HTML``、``JS``和``CSS``,采用``echarts``与``Jquery``框架获取[后端项目](https://github.com/wrm244/bigdata_depression)API实现可视化与页面动态设计。
